## React 简介

React 是一个用于构建用户界面的 JavaScript 库，由 Facebook 开发和维护。它的主要特点包括：

1. **组件化开发**
   - 可复用的组件
   - 声明式的编程方式
   - 单向数据流

2. **虚拟 DOM**
   - 高效的 DOM 操作
   - 智能的更新机制
   - 优秀的性能表现

3. **生态系统**
   - 丰富的第三方库
   - 完善的开发工具
   - 活跃的社区支持
