## 什么是 HTML？

HTML 是构建网页的基础，它使用标记标签来描述网页。HTML 文档包含了 HTML 标签和纯文本，这些文档也被称为网页。

HTML 标签通常成对出现，比如 `<p>` 和 `</p>`。第一个标签是开始标签，第二个标签是结束标签。开始和结束标签也被称为开放标签和闭合标签。

## 为什么学习 HTML？

- HTML 是网页开发的基础
- 它是所有前端技术的基石
- 掌握 HTML 有助于理解网页结构
- 对 SEO 优化至关重要
- 是响应式设计的基础
